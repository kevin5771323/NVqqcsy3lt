# Java考勤系统源码 java882

本项目是一款基于SpringBoot和Vue的考勤管理系统，提供完善的考勤功能，适用于企业内部员工考勤管理。

## 技术栈

- 前端：Vue + ElementUI
- 后端：SpringBoot + MyBatis
- 构建工具：Maven
- 数据库：MySQL 8.0
- 编译器推荐：IDEA、Eclipse

## 功能模块

### 普通员工功能

- 签到打卡：员工登录后可进行签到打卡
- 修改密码：员工可以修改个人密码
- 提交申请：员工可提交请假、固定资产申请
- 查看消息：员工可查看领导发布的会议通知和申请情况

### 中层领导功能

- 发布会议通知：领导可发布会议信息
- 任务审批：领导可审批员工申请
- 客户管理：管理客户信息，包括添加、删除、查看和修改

### 管理员功能

- 员工管理：员工入职、岗位调动等管理
- 部门管理：部门的新增和删除
- 财务管理：工资发放、导出考勤报表、查看考勤情况
- 查看统计信息：对员工及中层领导个人信息进行统计

## 系统角色

- 普通员工
- 中层领导
- 管理员

## 运行环境

- 前端：现代浏览器（推荐使用Chrome、Firefox等）
- 后端：Java 1.8及以上版本

## 部署步骤

1. 准备MySQL数据库环境
2. 导入数据库文件
3. 修改配置文件中的应用配置信息（如数据库连接信息）
4. 使用Maven构建项目
5. 部署前端和后端代码至服务器
